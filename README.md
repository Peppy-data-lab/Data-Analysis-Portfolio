# E-commerce Performance Overview (Jan–Feb 2025)

## Project Overview
This project analyzes an e-commerce dataset from January to February 2025 to uncover business performance insights, sales trends, customer behavior, and product performance. 

The original dataset was messy and required data cleaning before analysis could be carried out effectively.

## Objectives
- Clean and organize raw sales data
- Analyze sales performance
- Identify top-performing products
- Discover customer purchasing trends
- Generate business insights from the dataset

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning Techniques

## Data Cleaning Process
The raw dataset contained:
- Missing values
- Duplicate records
- Inconsistent formatting
- Incorrect data entries

These issues were cleaned and standardized before analysis.

## Key Insights
- Certain product categories generated the highest revenue
- Sales performance varied across different periods
- Customer purchasing patterns were identified
- Cleaned data improved reporting accuracy
